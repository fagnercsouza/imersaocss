Pra quem resolveu subir o site para o Github Pages agora, estava tudo lindo no Codepen, mas quando foi passar para os arquivos e salvar seu HTML e CSS usando o editor de código, seus estilos não aparecem? Veja as dicas:

  • Não esqueça de adicionar o link para o arquivo CSS na parte correta do HTML: ' '
  • No "caminho" do arquivo, os nomes de arquivos e de pastas têm que bater certinho. O código vê diferença entre minúsculas e maiúsculas! O exemplo acima diz que, junto com o arquivo index.html, existe uma pasta chamada css e, dentro dela, um arquivo chamado estilos.css. Sem o endereço/caminho da imagem o HTML não encontra o arquivo e o navegador não vai conseguir carregar os estilos. Se tiver dúvidas, volte no fim do vídeo da aula 1 que é quando fazemos isso no nosso projeto.
  • Se você usa Windows, algumas versões "escondem" a extensão do arquivo (por exemplo, .html ou .css) mas ela está lá! Confira, ao subir seus arquivos no GitHub, se por engano algum deles acabou ficando com o nome index.html.html ou estilos.css.css.
